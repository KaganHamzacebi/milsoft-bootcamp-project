# Milsoft Bootcamp Project

    Full Stack simple E-Commerce Project
Components:
- Category Listing
- Product Listing
- Cart / Cart Details
- Add Product to Cart
- Checkout Cart

- Redux / Redux-Thunk / Redux-Persist

Microservices:
